# ssimpul

## Local run with AI API

1. Install dependencies
   - `npm install`
2. Set OpenAI key
   - macOS/Linux: `export OPENAI_API_KEY=your_key`
3. Start server
   - `npm start`
4. Open `http://localhost:3000/portal.html`

The server provides `POST /api/ai/analyze` and the portal AI coach uses this endpoint by default.
